# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 4.x   | :white_check_mark: |
| 3.3.x   | :white_check_mark: |
| < 3.3   | :x:                |

## Reporting a Vulnerability

Security vulnerabilities may be reported through github issues for this project: https://github.com/kalm/kalm.js/issues
